Checking Composio...

Prove the connection passport handoff.

This demo loads connectable apps from Composio, creates an Agent Passport grant, gives that grant to a tiny agent, and runs one read-only provider tool without exposing raw OAuth tokens.

Run the demo

  • 1
    Provider configured Server has a Composio API key, but Agent Passport will not store tokens.
  • 2
    User connects selected app Composio owns OAuth, credential storage, and connected account id.
  • 3
    Passport grant is created The grant contains app, provider, status, scopes, and handoff reference.
  • 4
    Agent runs through provider The agent uses Composio to execute a read-only provider tool.

Composio apps

These come from the provider catalog. Each button asks Composio for that app's OAuth link.
