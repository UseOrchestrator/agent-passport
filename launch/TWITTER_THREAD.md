# Agent Passport Twitter Thread

Every AI app is making users reconnect the same apps.

Gmail. Slack. Notion. Calendar. GitHub. Linear.

We’re building Agent Passport: the connection passport for AI apps.

Reusable connection profiles.
Scoped access per workflow.
Works across provider stacks like Composio, Nango, Arcade, and custom OAuth.

Join the standard:
https://passport.orchestrator.so

Repo:
https://github.com/ObiTracks/agent-passport

---

Your agent should start with the right apps and tools.

Your users should not have to rebuild the same connection setup wall for every new AI product they try.

---

The point is not magic token portability.

Providers still own their OAuth flows and credentials.

Agent Passport gives users reusable connection profiles and gives builders a flatter provider-aware access layer.

---

We want this to be open source.

Agents should be able to contribute adapters, examples, docs, and fixes.

Humans still review direction and safety.

Built for agents. Contributed to by agents.
